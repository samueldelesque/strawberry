# Strawberry App

***FRONTEND***

- AngularJS
- Angular Route
- Angular Ressource
- Bower

***BACKEND***

- NodeJS
- npm
- Mongo
- Mongolian
- Restify


***To launch the app***

[https://github.com/Unitech/pm2]
> ```npm install```
> ```bower update```
> ```pm2 start node server/fileserver.js --name "Strawberry Fileserver"```
> ```pm2 start node server/api.js --name "Strawberry API"```
